# Model Card Template

Status: Stage 5A control template.  
Date: 2026-06-26.

This is not a completed model card.

## Required Fields

- model ID;
- model name;
- owner;
- version;
- purpose;
- decision supported;
- model type;
- inputs and source IDs;
- input hashes;
- assumptions;
- formula or algorithm;
- scenario controls;
- outputs;
- limitations;
- validation;
- sensitivity;
- uncertainty;
- reviewer;
- run log reference;
- status.

## No-Go Rule

No model output may be cited in OBC/FBC, consultation or statutory material unless a completed model card, run manifest, validation record and reviewer status exist.
