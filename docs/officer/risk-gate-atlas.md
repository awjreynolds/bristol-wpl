Status: officer and cabinet orientation guide.

## Purpose

This atlas explains where the main risks sit in the simulated workflow. It is a navigation aid, not a risk acceptance decision.

## How To Use This Atlas

1. Start with the gate area closest to the claim being made.
2. Check the linked issue, risk, gap and pitfall rows.
3. Confirm the next proof exists before drafting a positive claim.
4. If the proof is absent, write the claim as a blocker or condition.

## What This Atlas Must Not Be Used For

- It does not accept residual risk.
- It does not approve consultation, spend, procurement, OBC, FBC or statutory submission.
- It does not replace legal, finance, modelling, consultation, equalities, data-protection or officer review.
- It does not prove mitigation is complete.
