# Nottingham And Comparator Lessons

Status: officer-facing lessons control.  
Date: 2026-06-26.

This document records lessons only. It does not claim that Nottingham outcomes will happen in Bristol.

## Core Lesson

Nottingham is useful because it shows what an implemented WPL can require in practice: governance, employer support, licensing operations, parking controls, transport investment, monitoring and long-term public explanation.

The Bristol simulation must not copy Nottingham charge levels, elasticity, mode shift, congestion effects, parking-base assumptions or employer responses without Bristol evidence.
