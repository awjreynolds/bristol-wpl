## Main Findings

1. The no-go message is strong, but cold readers can still confuse control completeness with WPL readiness.
2. `GREEN` in the dashboard needs a visible legend explaining that it means control-only status.
3. `A` stages need a simple taxonomy: they create controls; they do not pass readiness gates.
4. Public/cabinet routes should be question-led, not folder-led.
5. Visual maps must avoid standalone `Complete` labels unless the same context says control-only and no readiness gate closes.
6. Agent sign-offs must remain simulation-only and must not be read as human officer, legal, finance, modelling, consultation or elected-member approval.
