# Source Note: SRC-BCC-0006

## What This Source Can Support

- Early feasibility-study context for a possible Bristol WPL scheme.
- Identification of scheme-development workstreams, project stages, dependencies and assumptions.
- Source material for questions that later OBC, consultation, implementation and operations work must test.

## What This Source Must Not Be Used To Claim

- That the feasibility assumptions are accepted as current OBC evidence.
- That revenue, boundary, parking-base or consultation assumptions are decision-grade.
- That Bristol has approved a scheme design or implementation programme.

## Source-Use Notes

Use this as a feasibility and assumption source. Any quantitative or scheme-design use needs refreshed evidence and reviewer sign-off.

Does not close ISS-0007.
