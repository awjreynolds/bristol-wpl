# Glossary

Controlled glossary. Expand abbreviations on first use in officer-facing documents.

| Term | Plain-English meaning | Controlled note |
|---|---|---|
| WPL | Workplace Parking Levy. A licensing scheme that can charge for workplace parking places if the statutory route is lawfully followed. | No Bristol WPL is approved by this repo. |
| OBC | Outline Business Case. A Five Case Model document used to test whether a proposal is strategically, economically, commercially, financially and managerially worth progressing. | No Bristol WPL OBC is assembled here. |
| FBC | Full Business Case. A later Five Case Model document that should support final investment and delivery decisions. | No Bristol WPL FBC is assembled here. |
| Five Case Model | UK public-sector business-case structure: Strategic, Economic, Commercial, Financial and Management cases. | Used as structure and assurance control only. |
| Scheme order | The legal instrument that would create, vary or revoke a WPL licensing scheme. | No Bristol scheme order is ready for reliance. |
| Secretary of State confirmation | Statutory confirmation route for an initial England-only WPL scheme under the controlled working assumption. | DfT engagement is not the same as confirmation. |
| WECA/MCA | West of England Combined Authority / Mayoral Combined Authority. | Role, consent/no-role and funding dependency remain unresolved. |
| DfT | Department for Transport. | Generic DfT guidance does not prove a WPL-specific confirmation route. |
| TAG | Transport Analysis Guidance. DfT guidance for transport appraisal. | Individual TAG units and model controls are still needed before appraisal reliance. |
| OAR | Options Assessment Report. | Not created here. |
| ASR | Appraisal Specification Report. | Not agreed here. |
| ASST | Appraisal Summary Specification Table or equivalent appraisal summary control used in this simulation. | Not completed here. |
| BCR | Benefit Cost Ratio. | No BCR exists here. |
| VFM | Value for Money. | No VFM category exists here. |
| Net proceeds | Levy income after lawful deductions that must be controlled against transport policy and programme requirements. | Finance and accounting controls are not complete. |
| CPZ/RPZ | Controlled Parking Zone / Residents' Parking Zone. | Used as potential displaced-parking mitigation, not as a selected Bristol measure. |
| EqIA | Equality Impact Assessment. | Existing EqIA material is evidence context only unless formally updated and reviewed. |
| DPIA | Data Protection Impact Assessment. | Not complete here. |
| P0 | Highest severity blocker. | Blocks relevant gates. |
| P1 | Serious blocker or controlled open issue. | Blocks unless a written condition, owner, deadline and residual risk are accepted. |
| Simulation sign-off | Agentic review result inside this repo. | Has no real-world legal, statutory, financial or professional effect. |
